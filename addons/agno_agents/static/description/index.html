Agno Agents Module for Odoo

This module provides integration with Agno AI agents within Odoo, allowing users to define, configure, and manage AI agents directly from the Odoo interface.

Features

  • Create and configure Agno agents with custom properties
  • Start and stop agents from Odoo interface
  • Agent status monitoring and management
  • Simple configuration for AI model settings
  • Agents accessible on configurable ports (default: 7777)
  • Support for OpenAI-like model APIs

Installation Requirements

Important: This module requires the Agno Python library to be installed in your Odoo environment.

Install Agno Library:

Run the following command to install the required Agno library:

pip install agno

Environment Variables:

Set up the required environment variable for API access:

export API_KEY="your-api-key-here"

System Requirements:

  • Python 3.8 or higher
  • Odoo 17.0 or higher
  • Internet connection for AI model API access

Usage

  1. Install the module in Odoo
  2. Navigate to "Agno Agents" menu
  3. Create a new agent with your desired configuration
  4. Set the API key environment variable
  5. Start the agent to make it accessible on the specified port

Configuration

Each agent can be configured with:

  • Agent Name: Unique identifier for your agent
  • Role: Description of the agent's purpose and capabilities
  • Model Settings: AI model ID, base URL, and API configuration
  • Advanced Options: History settings, debug mode, markdown support

Note: Agents run as separate processes and will be accessible via HTTP on the configured port.
