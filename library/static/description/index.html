Library Management System

A comprehensive library management module for Odoo 19 that helps you manage your book collection efficiently.

Features:

  • 📚 Book Management (Title, Author, Publish Date, ISBN)
  • 📋 List and Form Views for easy data entry
  • 🔍 Search and Filter capabilities
  • 🌐 JSON API Controller for external integrations
  • 🎯 User-friendly interface

Installation:

  1. Upload the module to your custom-addons directory
  2. Update the app list in Odoo
  3. Install the "Library Management" module from the Apps menu
  4. Access the Library menu from the main menu bar
